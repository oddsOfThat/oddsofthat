RFID controlled Spotify player

This is a spotify player I made that you control using RFID cards. Apologies for the low audio levels.
The hardware is

  • a raspberry pi 3
  • a 125khz rfid card reader
  • cheap usb powered speakers
  • a whole stack of RFID cards to store my favourite music

The software is mopidy (an open source music player which plays from storage or calls the Spotify API), a csv file mapping the RFID cards to the music resources and python scripts to hack it all together.
